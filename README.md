# Git & GitHub Learning Journey

This repository is dedicated to learning Git and GitHub from the scratch.
As a Computer Science student, I already use Git and GitHub regularly in my daily projects. Most of what I learned came naturally through building, collaborating, and solving real problems along the way.

Now, I want to deepen my understanding by learning the fundamentals properly not just how to use Git, but also how it actually works behind the scenes.

## Goals

- Understand core Git concepts deeply
- Learn Git commands from scratch
- Explore branching, merging, rebasing, and workflows
- Practice collaboration using GitHub
- Learn best practices for version control
- Build a stronger foundation for real-world development

## Topics Covered

- Git basics
- Repository management
- Commits and commit history
- Branching strategies
- Merge conflicts
- Remote repositories
- Pull requests
- GitHub workflows
- Open source contribution basics
